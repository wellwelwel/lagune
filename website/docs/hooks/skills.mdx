# `skills` hook

The `skills` hook lists the sub-skills **installed** in this project, with the tags that describe what each one covers. It is the deterministic listing the detect phase reads to find the sub-skills that fit, and you can run it yourself to see what is available here.

It lists what is present under `.bluespec/skills/`, the built-ins from the categories you installed plus any you authored with [`/bluespec.specialize`](/docs/commands/specialize). To see every category that exists, installed or not, use [`list --skills`](/docs/get-started/install#see-what-exists) instead.

## Run it

It takes no arguments. Each line is one sub-skill followed by its tags:

```bash
node ./.bluespec/hooks/skills.mjs
# => regex: Regular Expression [required]
#    network: Requests, Webhook, URL Fetch, Socket, IMDS, Redirect
#    ...
```

:::tip

- A `[required]` sub-skill ships a hook that locally scans the workspace.
- To actually run one of these against your code, import its file with a task, for example `@.bluespec/skills/<name>.md`. See [**sub-skills**](/docs/commands/skills) for the full walkthrough.

:::

# Slash Commands

Once **Lagune** is set up in your project, your **AI** agent unlocks a set of slash commands. There are two ways to use them, and both are governed by the **charter**: secure new work as you build it, or run the full audit over a project.

<Tabs>
  <TabItem value="build" label="⌨️ Build / Development" default>

You don't need to finish your work before securing it. Use [**`/lagune`**](/docs/commands/lagune) for the work you are writing **right now**:

| Command                                       | What it does for you                                                        |
| --------------------------------------------- | --------------------------------------------------------------------------- |
| [**/lagune.charter**](/docs/commands/charter) | Sets your project's security rules, the foundation `/lagune` builds against |
| [**/lagune**](/docs/commands/lagune)          | Builds what you ask and secures it as it is written, guided by the charter  |

:::tip[Use it anytime]
[**`/lagune`**](/docs/commands/lagune) honors your charter rules whenever it runs. It takes any prompt, builds what you asked, and makes it safe as it goes, not later.
:::

  </TabItem>
  <TabItem value="audit" label="🔬 Audit: Blue Team Flow">

The full **Blue Team flow**: read what a project does, map its risks, and close them. Each command builds on the previous, so following the list top to bottom is all it takes:

| #   | Command                                       | What it does for you                                                           |
| --- | --------------------------------------------- | ------------------------------------------------------------------------------ |
| 1   | [**/lagune.charter**](/docs/commands/charter) | Sets your project's security rules, proposed for you or shaped by what you say |
| 2   | [**/lagune.detect**](/docs/commands/detect)   | Reads your code and maps what your system does and where the risks are         |
| 3   | [**/lagune.plan**](/docs/commands/plan)       | Turns what detect found into a defense plan, with a fix for each finding       |
| 4   | [**/lagune.harden**](/docs/commands/harden)   | Applies the plan's fixes to your code, safely and one at a time                |
| 5   | [**/lagune.verify**](/docs/commands/verify)   | Proves each applied fix holds and closes out the ones that do                  |
